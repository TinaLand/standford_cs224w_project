# Figure Documentation

This directory contains documentation related to figure generation, requirements, and status.

## Contents

- **A_PLUS_FIGURES_SUMMARY.md** - Summary of all A+ grade required figures and their status
- **FIGURES_CHECKLIST.md** - Complete checklist of all figures with generation commands

## Figure Generation

All figures are generated using:
- `scripts/generate_report_figures.py` - Main figure generation script
- `scripts/create_additional_figures.py` - Additional analysis figures
- `scripts/check_and_generate_all_figures.py` - Verification and generation script

## Figure Location

All generated figures are stored in the `figures/` directory at the project root.

## Requirements

See [FIGURES_CHECKLIST.md](FIGURES_CHECKLIST.md) for complete requirements and status.
